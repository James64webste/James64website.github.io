Move with "W A S D"

Tip: if you get lost holding down "S" will eventualy take you back to the first room"

This is a remake of game I made for a computer science class a few years ago. The original was written from scratch in python, but since python can't be embedded into websites easily, I decided to remake it using bitsy. The main difference between this version and the original are the turtle graphics in python give a much better perspective effect than the tiles bitsy games use. Also this version is currently missing some things.
